Projects.

Brain Tumour Detection Using Matlab

Brain Tumor Analysis, provides a graphical user interface (GUI) for detecting and visualizing brain tumors in MRI images. The user can upload an MRI image, and upon clicking the "Detect Tumor" button, the program identifies and outlines the tumor region within the image. It utilizes image processing techniques such as binarization, morphological operations, and region labeling to isolate and highlight the tumor area. This tool can be useful for medical professionals and researchers in analyzing brain MRI scans for the presence of tumors.
GitHub Repository

Video Games Analysis

This project involves analyzing video game sales data to understand trends and make predictions. It includes steps like data preprocessing, visualization to gain insights, outlier treatment, encoding categorical variables, scaling features, and finally, building a predictive model using linear regression. The model achieves a high accuracy score, indicating its effectiveness in predicting global sales of video games. Additionally, the project explores different algorithms like Lasso regression and Decision Trees, with Decision Trees showing the best performance.
GitHub Repository

Airline Customer Satisfaction

This project involves analyzing customer satisfaction data from an airline, focusing on various factors such as gender, age, type of travel, class, and in-flight services. After preprocessing steps like handling missing values, outlier detection, and label encoding, a logistic regression model is built to predict customer satisfaction. The model achieves an accuracy of around 86% on the test data. The analysis includes visualizations like pie charts and bar plots to understand the distribution of satisfied and dissatisfied customers.
